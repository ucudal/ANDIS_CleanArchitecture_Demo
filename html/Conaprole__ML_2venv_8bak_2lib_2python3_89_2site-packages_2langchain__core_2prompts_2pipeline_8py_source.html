39 Prompt template for composing multiple prompt templates together.
40
41 This can be useful when you want to reuse parts of prompts.
42
43 A PipelinePrompt consists of two main parts:
44 - final_prompt: This is the final prompt that is returned
45 - pipeline_prompts: This is a list of tuples, consisting
