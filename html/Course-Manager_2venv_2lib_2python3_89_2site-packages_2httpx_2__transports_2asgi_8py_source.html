28class ASGITransport(httpcore.AsyncHTTPTransport):
29 """
30 A custom AsyncTransport that handles sending requests directly to an ASGI app.
31 The simplest way to use this functionality is to use the `app` argument.
32
33 ```
34 client = httpx.AsyncClient(app=app)
35 ```
36
37 Alternatively, you can setup the dispatch instance explicitly.
38 This allows you to include any additional configuration arguments specific
39 to the ASGITransport class:
40
41 ```
42 dispatch = httpx.ASGITransport(
43 app=app,
44 root_path="/submount",
45 client=("1.2.3.4", 123)
46 )
47 client = httpx.AsyncClient(dispatch=dispatch)
48 ```
49
50 Arguments:
51
52 * `app` - The ASGI application.
53 * `raise_app_exceptions` - Boolean indicating if exceptions in the application
54 should be raised. Default to `True`. Can be set to `False` for use cases
55 such as testing the content of a client 500 response.
56 * `root_path` - The root path on which the ASGI application should be mounted.
57 * `client` - A two-tuple indicating the client IP and port of incoming requests.
58 ```
59 """
