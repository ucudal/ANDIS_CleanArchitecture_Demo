7"""Notes about unicode handling in psutil
8======================================.
9
10Starting from version 5.3.0 psutil adds unicode support, see:
11https://github.com/giampaolo/psutil/issues/1040
12The notes below apply to *any* API returning a string such as
13process exe(), cwd() or username():
14
15* all strings are encoded by using the OS filesystem encoding
16 (sys.getfilesystemencoding()) which varies depending on the platform
17 (e.g. "UTF-8" on macOS, "mbcs" on Win)
18* no API call is supposed to crash with UnicodeDecodeError
19* instead, in case of badly encoded data returned by the OS, the
20 following error handlers are used to replace the corrupted characters in
21 the string:
22 * sys.getfilesystemencodeerrors() or "surrogatescape" on POSIX and
23 "replace" on Windows.
24
25For a detailed explanation of how psutil handles unicode see #1040.
26
27Tests
28=====
29
30List of APIs returning or dealing with a string:
31('not tested' means they are not tested to deal with non-ASCII strings):
32
33* Process.cmdline()
34* Process.cwd()
35* Process.environ()
36* Process.exe()
37* Process.memory_maps()
38* Process.name()
39* Process.net_connections('unix')
40* Process.open_files()
41* Process.username() (not tested)
42
43* disk_io_counters() (not tested)
44* disk_partitions() (not tested)
45* disk_usage(str)
46* net_connections('unix')
47* net_if_addrs() (not tested)
48* net_if_stats() (not tested)
49* net_io_counters() (not tested)
50* sensors_fans() (not tested)
51* sensors_temperatures() (not tested)
52* users() (not tested)
53
54* WindowsService.binpath() (not tested)
55* WindowsService.description() (not tested)
56* WindowsService.display_name() (not tested)
57* WindowsService.name() (not tested)
58* WindowsService.status() (not tested)
59* WindowsService.username() (not tested)
60
61In here we create a unicode path with a funky non-ASCII name and (where
62possible) make psutil return it back (e.g. on name(), exe(), open_files(),
63etc.) and make sure that:
64
65* psutil never crashes with UnicodeDecodeError
66* the returned path matches
67"""

101def try_unicode(suffix):
102 """Return True if both the fs and the subprocess module can
103 deal with a unicode file name.
104 """
105 sproc = None
106 testfn = get_testfn(suffix=suffix)
107 try:
108 safe_rmpath(testfn)
109 create_py_exe(testfn)
110 sproc = spawn_testproc(cmd=[testfn])
111 shutil.copyfile(testfn, testfn + '-2')
112 safe_rmpath(testfn + '-2')
113 except (UnicodeEncodeError, OSError):
114 return False
115 else:
116 return True
117 finally:
118 if sproc is not None:
119 terminate(sproc)
120 safe_rmpath(testfn)
